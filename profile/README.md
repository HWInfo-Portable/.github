If you are tired of heavyweight installers or separate tools for each component, **HWInfo Portable** is for you.

## Quick Start with HWInfo Portable

1. **Get HWInfo Portable** — Download the portable `.exe` (no installation required). No admin rights needed.
2. **Launch** — Double-click `HWInfo_Portable.exe`. The main sensor window opens in under 3 seconds.
3. **Start monitoring instantly** — By default, **HWInfo Portable** shows CPU temperatures, clock speeds, and fan RPM. Resize any column.
4. **Enable detailed sensors** — Click "Sensors" → "Configure" → select voltages, power draw, drive temperatures, or network activity.
5. **Log to file** — Go to "Logging" → "Start logging". **HWInfo Portable** saves CSV files for later analysis.
6. **Adjust alerts (optional)** — Set temperature or voltage thresholds. **HWInfo Portable** plays a sound or shows warnings.
7. **Exit cleanly** — Close the window. **HW Info Portable** leaves no processes, no registry keys, and no startup entries.

## Understanding HWInfo Portable Core Components

**HWInfo Portable** is not a simple thermometer. It combines several essential diagnostic systems:

- **Kernel-level sensor driver (on-demand)** — safely accesses CPU, GPU, motherboard, and disk sensors without permanent installation.
- **Real-time sensor graphs** — plot temperature, frequency, voltage, fan speed, and power consumption over time.
- **Summary system report** — one-click export of all hardware specifications: BIOS version, memory timings, PCIe link speeds.
- **Remote monitoring** — serve sensor data via lightweight web interface (optional, disabled by default).
- **Alerting system** — per-sensor minimum/maximum thresholds with visual or audio notifications.
- **Logging and benchmarking** — continuous CSV logging and timed benchmark runs.

All these work together through one portable executable. You never install separate drivers or kernel modules permanently.
